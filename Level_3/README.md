# Assignment 3: Sign Up Page Layout

## Task
 
Build a clean, responsive **Sign Up Page** with modern UI design and illustration containing:
- Left column with background color and image/illustration
- Form fields (text, email, and password)

---

## Focus Areas in Design

- Center alignment
- Clean two-column layout with proper spacing
- Flat-style illustration integration

---

## What Developers Will Learn

- CSS layout and styling using Flexbox
- Handling input hover and focus states
- Applying consistent spacing and typography
- Image handling and responsive design basics

---

## Technologies Used

- **HTML5** – Semantic structure of the signup page  
- **CSS3** – Styling, layout, and responsiveness  
- **Flexbox** – Two-column layout and center alignment  
- **Media Queries** – Responsive behavior across screen sizes  

---

## Steps Followed to Build the Project

### 1. HTML Structure
- Created a main container to hold the full-page layout.
- Divided the layout into two sections:
  - Illustration section (left column)
  - Form section (right column)
- Used clear and meaningful class names for better readability and maintenance.

---

### 2. Two-Column Layout Using Flexbox
- Implemented a side-by-side layout for desktop screens.
- Ensured both columns occupy equal height of the viewport.
- Maintained center alignment of content inside each column.

---

### 3. Illustration Section Design
- Added a visually appealing background color and illustration.
- Centered the illustration and accompanying text.
- Ensured the image scales correctly on different screen sizes.
- Maintained clean spacing between image and text content.

---

### 4. Form Section Layout
- Centered the form vertically and horizontally for better focus.
- Used a constrained width to improve readability.
- Clearly separated form header, input fields, and footer.

---

### 5. Form Fields and Input States
- Added input fields for full name, email address, and password.
- Implemented hover states to indicate interactivity.
- Implemented focus states to improve usability and accessibility.
- Maintained consistent styling across all input fields.

---

### 6. Call-to-Action Button
- Designed a clear and prominent primary “Sign Up” button.
- Used visual feedback for hover and active states.
- Ensured the button stands out as the main action.

---

### 7. Typography and Spacing
- Applied font hierarchy for headings, labels, and helper text.
- Used consistent margins and padding throughout the layout.
- Ensured good readability and visual balance.

---

### 8. Responsive Design
- Adapted the two-column layout to a single-column layout on smaller screens.
- Reduced image size and padding on mobile devices.
- Ensured the page remains usable and visually clean across all screen sizes.

---

## Key Concepts Used

- Semantic HTML structure
- Flexbox for layout and alignment
- Input hover and focus state handling
- Consistent spacing and typography
- Responsive design principles

---

## Assignment Outcome

- Clean and modern signup page design
- Fully responsive two-column layout
- Proper illustration and form integration
- Clear input states and call-to-action
- User-friendly and professional UI
